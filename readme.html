# Application E-Commerce pour Équipements Électroniques ## Description Cette application e-commerce est conçue pour faciliter la vente d'équipements électroniques, tels que des ordinateurs portables, des montres, des téléphones et d'autres gadgets technologiques. Développée avec WooCommerce et WordPress, elle offre une expérience utilisateur fluide et intuitive, permettant aux clients de naviguer facilement dans le catalogue de produits, d'ajouter des articles à leur panier et de finaliser leurs achats en toute sécurité. ## Fonctionnalités - Catalogue complet d'équipements électroniques - Recherche avancée et filtres de produits - Processus de commande simplifié - Suivi des commandes - Support client intégré ## Instructions d'Installation Pour installer et configurer l'application e-commerce, suivez ces étapes : ### Prérequis - Un serveur web compatible avec PHP (version 7.4 ou supérieure) - Une base de données MySQL (version 5.6 ou supérieure) ### Étapes d'Installation 1. Télécharger WordPress - Rendez-vous sur le site officiel de [WordPress](https://wordpress.org/download/) et téléchargez la dernière version. 2. Télécharger WooCommerce - Installez le plugin WooCommerce depuis le répertoire des plugins WordPress ou téléchargez-le depuis [WooCommerce](https://woocommerce.com/products/woocommerce/). 3. Configurer le Serveur Web - Décompressez les fichiers WordPress dans le répertoire racine de votre serveur. - Créez une base de données MySQL pour l'application. 4. Installer WordPress - Accédez à votre nom de domaine ou à l'URL locale où vous avez installé WordPress. - Suivez les instructions à l'écran pour configurer WordPress (choisissez votre langue, entrez les détails de la base de données, etc.). 5. Activer WooCommerce - Une fois WordPress installé, accédez au tableau de bord. - Allez dans "Extensions" > "Ajouter" et recherchez "WooCommerce". - Installez et activez le plugin. 6. Configurer WooCommerce - Suivez l'assistant de configuration pour définir les paramètres de base (monnaie, méthodes de paiement, options d'expédition). 7. Ajouter des Produits - Dans le tableau de bord, allez dans "Produits" > "Ajouter un produit" pour commencer à ajouter vos équipements électroniques. 8. Personnaliser le Thème - Choisissez un thème compatible avec WooCommerce et personnalisez-le selon vos besoins. ## Importer la base de données 1. Téléchargez le fichier `database/ecommerce.sql`. 2. Importez-le dans votre gestionnaire de base de données (ex. : phpMyAdmin ou MySQL) : ```bash mysql -u [utilisateur] -p [nom_de_la_base] < database/ecommerce.sql ## Identifiants Administrateur Principal Après l'installation, utilisez les identifiants suivants pour accéder au tableau de bord en tant qu'administrateur : - Nom d'utilisateur : Monya_Oubella - Mot de passe : Monya_Admin1234@ Note : Il est fortement recommandé de changer le mot de passe après la première connexion pour des raisons de sécurité.
